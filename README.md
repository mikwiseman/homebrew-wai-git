# homebrew-wai-git

Homebrew tap for [acp](https://github.com/mikwiseman/wai-git) - AI-powered git add, commit, and push.

## Installation

```bash
brew tap mikwiseman/wai-git
brew install acp
```

## Usage

```bash
acp --help
```
